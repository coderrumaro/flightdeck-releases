# Flightdeck Releases

Public download page for **Flightdeck** desktop installers.

This repository contains **release binaries only** — not the application source code.

## Download

Open [Releases](https://github.com/coderrumaro/flightdeck-releases/releases) for the latest Windows (.exe / .msi) and macOS (.dmg) installers.

Installers are unsigned; your OS may show a one-time security prompt on first run.
